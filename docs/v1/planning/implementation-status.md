This matrix tracks implementation against the final V1 core-pipeline acceptance areas. It is an implementation aid, not a substitute for working code, tests, docs, review, and pushed commits.

Status values:

- Not started
- In progress
- Implemented
- Tested
- Documented
- Reviewed
- Done
