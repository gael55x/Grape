## Alpha.2 Pre-Beta Status

This note captures the real pre-beta state for human review. The detailed matrix below remains the implementation-area tracker.

Done:

- `grape-context@0.1.0-alpha.2` is published and the release exists.
- The context transport wedge is proven through CLI and MCP `grape_get_context`.
- MCP stdio smoke exists.
- Same-session two-turn omission works when task/session identity is stable.
- `PINNED`, `OMIT_UNCHANGED`, `RESTORE_AVAILABLE`, and `INVALIDATE_PREVIOUS` are implemented in the current transport slice.
- Branch/dependency invalidation, explicit reset invalidation, and omitted restore are proven by behavior coverage.
- The external benchmark workspace has a 13/13 scripted pass when run with the documented methodology and stable task/session contract.

Now:

- Agent session contract, README links, roadmap/status refresh, npm-cache note, install/connect clarity, and seamless happy-path docs.
- External benchmark workspace dependency alignment to alpha.2 after approval, without changing benchmark methodology.

Next:

- Clearer task/session mismatch UX.
- Session reset fixture benchmark.
- MCP install smoke against the global package inside this repo's gate.
- Restore-path golden tests.
- Optional alpha.3 after docs/checks land.

Soon:

- Better TypeScript compiler retrieval and exact-span ranking.
- Grape-observed command/test recording.
- Stronger trust/proof depth, broader fixtures, and better dirty-worktree/session handling.

Later:

- Embeddings, Rust, cloud/team sync, full graph extraction, memory-platform features, IDE plugins, and stable `1.0.0`.

See [`beta-readiness.md`](beta-readiness.md) for the review checklist and benchmark workspace alignment note.
